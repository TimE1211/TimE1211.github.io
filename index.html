Drawn Together

DrawnTogether was my course's final project. I used most of what I learned in this app and more. DrawnTogether is an app that allows two people to draw together using an API built completely in swift. I built the API from scratch and currently host it on my own device. The drawing portion of this app is relatively simple, but the API portion was extremely complicated. Learning on how API's work and how to use them when made in swift was a hard, but valuable experience. I used PerfectlySoft's Library to make the API, a canvas view and draw functions for the drawing portion, Almofire and SwiftyJSON to make the Network interactions a little bit easier and faster to write, and SQLite to store the data in my API.

Celluoid

Celluoid was one of my last projects during my course. This was an app to show Movies that are out currently in movie theatures. There were three types of movies being shown on my app: Now Playing movies, Popular movies, and movies that you decided to add to your own Watchlist. I used tab bars and gliding collection views in this app.

Forecaster

Forecaster was one of the first apps I made. It is a simple weather app that I made for myself to display the basics of the weather. It shows weather conditions, what you should bring with you for the days weather, the date for that weather, can show weekly weather, and can show weather for different locations. I used the darksky API, coreLocation, and geoLocation in this app.
